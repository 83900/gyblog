🌟 项目简介

Roo-Code 是一个开源的 AI 编码助手，集成在 VS Code 中。它能理解你的整个代码库，自动编写、修改和测试代码。作为 Cursor 的开源替代品，Roo-Code 支持多种 AI 模型，包括 Claude、GPT-4、Gemini 等。

🔥 为什么选它而不是 Cursor？

1. 完全免费开源

Apache 2.0 许可，源代码完全开放，不用担心突然收费或者停止服务。

2. 多模型支持

不绑定任何 AI 厂商，可以自由切换 Claude、GPT-4、Gemini、本地模型等。

3. 隐私优先

支持本地模型（如 Ollama），代码完全不离开你的电脑，不用担心数据泄露。

4. 全代码库理解

不只是单文件编辑，能理解整个项目结构，自动关联相关文件。

📦 安装方法

# 在 VS Code 扩展市场安装

1. 打开 VS Code

2. 按 Ctrl+Shift+X 打开扩展面板

3. 搜索 "Roo-Code"

4. 点击安装

# 或者命令行安装

code --install-extension RooVeterinaryInc.roo-cline

⚙️ 配置 AI 模型

方案一：使用 Anthropic Claude（推荐）

1. 访问 console.anthropic.com 获取 API Key
2. 在 Roo-Code 设置中选择 "Anthropic"
3. 填入 API Key
4. 选择模型：claude-3-5-sonnet（性价比最高）

方案二：使用本地模型（完全免费）

# 1. 安装 Ollama

brew install ollama # macOS

# 2. 下载模型

ollama pull codestral

# 3. 在 Roo-Code 中配置

Provider: Ollama

Base URL: http://localhost:11434

Model: codestral

💡 实际使用示例

场景 1：创建新功能

你在聊天框输入：
"创建一个 React 登录组件，包含邮箱和密码输入框，有表单验证，使用 Tailwind CSS"

Roo-Code 会：
✅ 自动创建 Login.jsx 文件
✅ 编写完整的组件代码
✅ 添加表单验证逻辑
✅ 应用 Tailwind 样式

场景 2：修复 Bug

选中报错的代码，然后说：
"这段代码报错 'Cannot read property of undefined'，帮我修复"

Roo-Code 会：
✅ 分析错误原因
✅ 定位问题代码
✅ 给出修复方案
✅ 自动应用修改

场景 3：代码重构

"这个函数太长了，帮我拆分成几个小函数"

Roo-Code 会：
✅ 分析函数职责
✅ 按功能拆分
✅ 保持接口不变
✅ 更新所有调用点

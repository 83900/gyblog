作为一个经常和论文打交道的人，我曾经也是 Word 的忠实用户——直到某次改参考文献格式改了整整一下午，才下定决心学 LaTeX。入门的时候确实痛苦，对着一堆反斜杠和花括号一脸懵，但撑过前三天之后，回头看 Word 排版论文的那种体验，就像从智能手机回到翻盖手机一样。

这篇教程是我在 Mac 和 Windows 上反复折腾后总结出来的部署指南。不管你是理工科写论文、还是单纯想让文档排版更专业，跟着走一遍，半小时就能跑起来。

⚠️ ATTENTION

安装 LaTeX 发行版大约需要 4GB 磁盘空间。如果网络环境受限，可以使用 MiKTeX（基础版仅 200MB，按需下载宏包）。

Step 1 — 为什么要用 LaTeX？

很多人第一次听到 LaTeX 的反应是："Word 不够用吗？" 答案取决于你要写什么。

LaTeX 的优势：

  1. 排版质量碾压 Word — LaTeX 由专业排版引擎驱动，公式、参考文献、目录的生成都是全自动的。你写论文时不用手动调格式，引擎帮你处理一切。
  2. 数学公式原生支持 — Word 里插入公式需要点菜单、用鼠标选符号。LaTeX 里直接写 $E = mc^2$，又快又好看。
  3. 参考文献管理自动化 — 配合 BibTeX，引用文献只需要一个命令 \cite{key}，列表自动生成。
  4. 版本控制友好 — LaTeX 源文件是纯文本，可以用 Git 管理版本、多人协作。
  5. 跨平台一致性 — 同样的 .tex 文件，在 Mac、Windows、Linux 上编译结果完全一致。
  6. 学术界事实标准 — 绝大多数理工科期刊、会议论文都要求或推荐 LaTeX 格式。

LaTeX 的弊端：

  1. 学习曲线陡峭，入门需要记住基本语法
  2. 不是"所见即所得"，需要编译后才能看到效果
  3. 图形化排版（拖拽图片位置）比较困难
  4. 协作门槛高，团队里有人不会就很难配合

一句话总结：写论文、技术文档 → 用 LaTeX。写简历、简单报告 → 用 Word。

Step 2 — 下载 LaTeX 发行版

LaTeX 本身是一套排版语言，需要一个「发行版」来提供编译器和宏包。就像 Python 需要解释器一样。

发行版 平台 大小 推荐
TeX Live 全平台 ~4GB 跨平台首选
MacTeX 仅 Mac ~4GB Mac 用户专用
MiKTeX 全平台 ~200MB 轻量级，按需下载

下载地址：

  • TeX Live（Windows/Linux）：https://www.tug.org/texlive/
  • MacTeX（Mac）：https://www.tug.org/mactex/
  • MiKTeX：https://miktex.org/download

安装建议：

  • Mac 用户直接下载 MacTeX，安装包自带 TeX Live + 几个 GUI 工具
  • Windows 用户推荐 TeX Live（稳定完整）或 MiKTeX（轻量）
  • 安装时选择「完整安装」，避免后面缺宏包

Step 3 — 安装 TeXStudio 编辑器

发行版只是编译器，写代码还需要一个编辑器。推荐 TeXStudio —— 免费、开源、跨平台、功能强大。

下载地址：https://texstudio.org/

Windows 安装：

  1. 下载 texstudio-x.x.x-win-qt6.exe
  2. 双击安装，一路 Next
  3. 首次打开会自动检测已安装的 TeX Live / MiKTeX
  4. 验证：Help → Check LaTeX Installation，应显示绿色对勾

Mac 安装：

  1. 下载 texstudio-x.x.x-osx.dmg
  2. 拖入 Applications 文件夹
  3. 首次打开可能需要在系统设置中允许

TeXStudio 基础配置：

  • 默认编译器设为 XeLaTeX（支持中文）：Options → Build → Default Compiler
  • 开启实时预览：Options → Internal PDF Viewer → 勾选实时更新
  • 中文界面：Options → General → Language → 中文

测试第一个文档（按 F5 编译）：

\documentclass{article} \usepackage[UTF8]{ctex} \title{我的第一篇 LaTeX 文档} \author{Guyi} \date{\today} \begin{document} \maketitle Hello, \LaTeX! 这是我的第一篇文章。 \end{document}

如果右侧 PDF 预览区成功显示内容，环境搭建成功！

Step 4 — AI 提示词 + 命令行编译

用 AI 帮你写 LaTeX： 以下是几个万能 prompt：

Prompt 1 — 生成完整论文模板：

请帮我写一篇 IEEE 会议论文的 LaTeX 模板，包含：
- 标题、作者、摘要
- 三个章节（引言、方法、实验）
- 一个表格、一个公式、一张图片的引用
- 参考文献（使用 BibTeX）
使用中文撰写。

Prompt 2 — 公式转 LaTeX：

请将以下数学公式转换为 LaTeX 代码：
勾股定理、泰勒展开、高斯积分、贝叶斯公式

Prompt 3 — 表格生成：

请用 LaTeX 生成一个 5 行 4 列的表格，内容如下：
姓名 | 年龄 | 专业 | 成绩
要求带边框、居中对齐、首行加粗。

命令行快速编译：

# 基本编译 xelatex myfile.tex # 编译两次（解决引用和目录） xelatex myfile.tex && xelatex myfile.tex # 自动判断编译次数 latexmk -xelatex myfile.tex # 清理中间文件 latexmk -c myfile.tex # 一键编译并打开（Mac） xelatex myfile.tex && open myfile.pdf

Step 5 — 进阶：编译器区别与实用指令

pdflatex vs xelatex：

特性 pdflatex xelatex
中文支持 需额外配置 原生支持
字体支持 仅 TeX 内置 可使用系统字体
Unicode 部分支持 完全支持
适用场景 纯英文文档 中文 / 多语言

结论：直接用 xelatex，中文英文都通吃。

中文 LaTeX 最小模板：

\documentclass[UTF8]{ctexart} \begin{document} 这是一段中文。English text is also OK. \end{document}

实用指令速查表：

功能 指令
加粗\textbf{文字}
斜体\textit{文字}
行内公式$E = mc^2$
行间公式\[ \int_0^\infty e^{-x} dx = 1 \]
插入图片\includegraphics[width=0.8\textwidth]{img.png}
引用文献\cite{bibkey}
代码高亮\usepackage{listings}

推荐宏包：

\usepackage{ctex} % 中文支持 \usepackage{graphicx} % 插入图片 \usepackage{amsmath} % 数学公式增强 \usepackage{hyperref} % 超链接 \usepackage{listings} % 代码高亮 \usepackage{geometry} % 页面边距 \usepackage{booktabs} % 专业表格

💡 总结

LaTeX 的学习曲线确实比 Word 陡，但一旦上手，排版效率和质量都会大幅提升。加上 AI 工具的辅助，入门门槛已经大大降低。

🌟 项目简介

DeepSeek-R1 是由深度求索（DeepSeek）开发的开源推理大模型，专注于复杂推理任务。它在数学推理、代码生成、逻辑推理等任务上达到了与顶级闭源模型相媲美的水平。

最让我印象深刻的是，DeepSeek-R1 在 AIME 2024 数学竞赛中达到了 79.8% 的准确率，几乎追平了 OpenAI o1-1217 的表现。要知道，这可是开源模型！

💡 核心技术亮点

1. 纯强化学习训练

DeepSeek-R1-Zero 版本完全通过强化学习训练，没有使用任何监督微调数据。这证明了仅通过 RL 就能让模型自发产生强大的推理能力。

2. 多阶段训练框架

DeepSeek-R1 在此基础上加入了冷启动数据和多阶段训练，使模型输出更加清晰、结构化，同时保持强大的推理能力。

3. 知识蒸馏

DeepSeek-R1 的推理能力可以蒸馏到更小的模型中。蒸馏后的 7B 模型在数学推理上甚至超过了 GPT-4o，而 1.5B 模型也能在某些任务上匹敌 o1-mini。

📊 性能对比

数学推理：
• AIME 2024：79.8% pass@1（接近 o1-1217 的 79.2%）
• MATH-500：97.3%（超过 o1-1217 的 96.4%）

编程能力：
• Codeforces：2029 Elo rating（超过 96.3% 的人类参与者）
• SWE-bench Verified：49.2%（开源模型中最佳）

知识问答：
• MMLU：90.8%
• GPQA Diamond：71.5%

🚀 快速上手

# 通过 Ollama 本地部署

ollama run deepseek-r1

# 选择不同大小的模型

ollama run deepseek-r1:1.5b # 轻量版

ollama run deepseek-r1:7b # 平衡版

ollama run deepseek-r1:32b # 高性能版

# 使用 API 调用

curl https://api.deepseek.com/v1/chat/completions \

-H "Authorization: Bearer YOUR_API_KEY" \

-d '{"model": "deepseek-reasoner", "messages": [{"role": "user", "content": "解释量子纠缠"}]}'

💡 实际应用场景

场景 1：数学辅导

DeepSeek-R1 可以作为 AI 数学老师，不仅给出答案，还能展示完整的解题思路和推理过程。

场景 2：代码审查

让 DeepSeek-R1 分析你的代码，它能找到潜在的 bug 并给出优化建议，推理过程清晰可追溯。

场景 3：逻辑推理

处理复杂的逻辑问题时，DeepSeek-R1 的思维链推理能力让它能够逐步分析，给出准确的结论。

💡 提示：DeepSeek-R1 在处理需要多步推理的任务时表现最佳。简单的问答任务，使用 DeepSeek-V3 可能更高效。

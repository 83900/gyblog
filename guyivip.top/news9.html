OpenAI Codex CLI：终端里的 AI 编程助手

April 20, 2026

OpenAI Codex CLI 简介

OpenAI Codex CLI 是 OpenAI 推出的轻量级终端编程助手，用 Rust 编写，直接在你的命令行中运行。它能理解你的代码库、执行代码修改、运行测试，就像一个坐在你身边的结对编程伙伴。

核心亮点

- 76,000+ GitHub Stars，Apache-2.0 开源协议
- Rust 编写，启动快、内存占用低
- 支持 ChatGPT 订阅直接登录使用
- 可以读取、修改代码，执行 shell 命令
- 支持 VS Code、Cursor 等 IDE 集成

安装方式

Quickstart: 

# 使用 npm 安装
npm install -g @openai/codex

# 或使用 Homebrew
brew install --cask codex

# 启动
codex
核心功能

- 代码理解：读取整个代码库，理解项目结构
- 代码修改：根据指令自动修改代码文件
- 测试运行：自动执行测试并修复失败用例
- Git 操作：自动处理 git 工作流（提交、分支等）
- 终端交互：直接在终端中完成所有操作

适用场景

- 开发者日常编码辅助
- 代码重构和迁移
- Bug 修复和调试
- 快速原型开发

与 Claude Code 的区别

Codex CLI 和 Claude Code 是同类产品，都致力于在终端中提供 AI 编程助手体验。Codex 更加轻量，适合喜欢命令行工作流的开发者。

GitHub: https://github.com/openai/codex

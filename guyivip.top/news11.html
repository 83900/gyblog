一、为什么要用 LaTeX？

LaTeX vs Word：优劣对比

很多人第一次听到 LaTeX 的反应是："Word 不够用吗？" 答案取决于你要写什么。

LaTeX 的优势

  1. 排版质量碾压 Word
    LaTeX 由专业排版引擎驱动，公式、参考文献、目录的生成都是全自动的。你写论文时不用手动调格式，引擎帮你处理一切。
  2. 数学公式是原生支持
    Word 里插入公式需要点菜单、用鼠标选符号。LaTeX 里直接写 $E = mc^2$，又快又好看。
  3. 参考文献管理自动化
    配合 BibTeX，引用文献只需要一个命令 \cite{key}，参考文献列表自动生成，格式自动适配期刊要求。
  4. 版本控制友好
    LaTeX 源文件是纯文本，可以用 Git 管理版本、多人协作。Word 的 .docx 是二进制文件，diff 基本不可用。
  5. 跨平台一致性
    同样的 .tex 文件，在 Mac、Windows、Linux 上编译结果完全一致。不存在"我电脑上排版好的，你打开就乱了"的问题。
  6. 学术界事实标准
    绝大多数理工科期刊、会议论文都要求或推荐 LaTeX 格式投稿。学会它是迟早的事。

LaTeX 的弊端

  1. 学习曲线陡峭
    入门需要记住基本语法，排版问题调试起来比 Word 麻烦得多。
  2. 实时预览缺失
    不像 Word "所见即所得"，LaTeX 需要编译后才能看到效果（虽然现代编辑器有实时预览）。
  3. 图形化排版困难
    如果需要自由拖拽图片位置、做海报或图文混排，Word / PowerPoint 更直观。
  4. 协作门槛高
    团队里有人不会 LaTeX，协作就变成你一个人的事。

一句话总结

写论文、技术文档 → 用 LaTeX。写简历、简单报告 → 用 Word。 两者不是替代关系，是不同场景的最佳选择。

二、下载 LaTeX 发行版

LaTeX 本身是一套排版语言，需要一个「发行版」来提供编译器和宏包。就像 Python 需要解释器一样。

两大主流发行版

发行版平台大小推荐
TeX LiveWindows / Linux / Mac~4GB（完整版）跨平台首选
MacTeX仅 Mac~4GBMac 用户专用（基于 TeX Live）
MiKTeXWindows / Mac / Linux~200MB（基础版）轻量级，按需下载宏包

下载地址

  • TeX Live（Windows/Linux）：https://www.tug.org/texlive/
  • MacTeX（Mac）：https://www.tug.org/mactex/
  • MiKTeX：https://miktex.org/download

安装建议

  • Mac 用户：直接下载 MacTeX，安装包自带 TeX Live + 几个 GUI 工具，一键搞定。
  • Windows 用户：推荐 TeX Live（稳定、完整）或 MiKTeX（轻量、按需下载）。
  • 安装时选择「完整安装」，避免后面缺宏包。

三、安装编辑器：以 TeXStudio 为例

发行版只是编译器，写代码还需要一个编辑器。推荐 TeXStudio —— 免费、开源、跨平台、功能强大。

下载 TeXStudio

https://texstudio.org/

安装步骤

Windows

  1. 下载 texstudio-x.x.x-win-qt6.exe
  2. 双击安装，一路 Next
  3. 首次打开 TeXStudio，它会自动检测已安装的 TeX Live / MiKTeX
  4. 验证：菜单栏 → Help → Check LaTeX Installation，应该显示绿色对勾

Mac

  1. 下载 texstudio-x.x.x-osx.dmg
  2. 拖入 Applications 文件夹
  3. 首次打开可能需要在 系统偏好设置 → 安全性与隐私 中允许
  4. 同样通过 Help → Check LaTeX Installation 验证

TeXStudio 基础配置

  • 设置默认编译器：Options → Configure TeXStudio → Build → Default Compiler → 选择 XeLaTeX（支持中文）
  • 开启实时预览：Options → Configure TeXStudio → Internal PDF Viewer → 勾选实时更新
  • 中文界面：Options → Configure TeXStudio → General → Language → 中文（Chinese）

测试第一个 LaTeX 文档

打开 TeXStudio，输入以下内容，按 F5 编译：

\documentclass{article}
\usepackage[UTF8]{ctex}
\title{我的第一篇 LaTeX 文档}
\author{Guyi}
\date{\today}
\begin{document}
\maketitle
Hello, \LaTeX! 这是我的第一篇文章。
\end{document}

如果右侧 PDF 预览区成功显示标题和内容，恭喜你，LaTeX 环境搭建成功！

四、AI 提示词 + 命令行快速编译

用 AI 帮你写 LaTeX

现在有了 ChatGPT / Claude，写 LaTeX 变得极其简单。以下是几个万能 prompt：

Prompt 1：生成完整论文模板

请帮我写一篇 IEEE 会议论文的 LaTeX 模板，包含：
- 标题、作者、摘要
- 三个章节（引言、方法、实验）
- 一个表格、一个公式、一张图片的引用
- 参考文献（使用 BibTeX）
使用中文撰写。

Prompt 2：公式转 LaTeX

请将以下数学公式转换为 LaTeX 代码：
勾股定理、泰勒展开、高斯积分、贝叶斯公式

Prompt 3：表格生成

请用 LaTeX 生成一个 5 行 4 列的表格，内容如下：
姓名 | 年龄 | 专业 | 成绩
要求带边框、居中对齐、首行加粗。

Prompt 4：排版调试

我的 LaTeX 编译报错：Undefined control sequence \xxxxx
请帮我分析原因并给出修复方案。完整代码如下：
[粘贴你的 .tex 代码]

命令行快速编译

不打开 GUI 编辑器，直接在终端编译：

# 基本编译（生成 PDF）
xelatex myfile.tex

# 编译两次（解决引用和目录）
xelatex myfile.tex
xelatex myfile.tex

# 使用 latexmk 自动判断编译次数
latexmk -xelatex myfile.tex

# 清理中间文件
latexmk -c myfile.tex

# 一键编译并打开 PDF（Mac）
xelatex myfile.tex && open myfile.pdf

# 一键编译并打开 PDF（Windows PowerShell）
xelatex myfile.tex; Start-Process myfile.pdf

五、进阶内容

pdflatex vs xelatex：该用哪个？

特性pdflatexxelatex
中文支持需额外配置原生支持（配合 ctex 宏包）
字体支持仅限 TeX 内置字体可直接使用系统字体
Unicode部分支持完全支持
编译速度稍快稍慢（可忽略）
适用场景纯英文文档中文文档 / 多语言文档

结论：直接用 xelatex，中文英文都通吃。

中文 LaTeX 编写规则

最小中文文档模板

\documentclass[UTF8]{ctexart}
\begin{document}
这是一段中文。English text is also OK.
\end{document}

中英文混排注意事项

  • 使用 ctex 宏包或 ctexart 文档类
  • 编译器必须用 xelatex，不要用 pdflatex
  • 中英文之间自动加空格：\usepackage{xeCJK} 会自动处理
  • 引号：``中文引号'' → "中文引号"
  • 破折号：直接输入中文破折号即可

推荐宏包

\usepackage{ctex}          % 中文支持
\usepackage{graphicx}      % 插入图片
\usepackage{amsmath}       % 数学公式增强
\usepackage{hyperref}      % 超链接
\usepackage{listings}      % 代码高亮
\usepackage{geometry}      % 页面边距设置
\usepackage{booktabs}      % 专业表格（三线表）
\usepackage{cite}          % 参考文献优化

总结

LaTeX 的学习曲线确实比 Word 陡，但一旦上手，排版效率和质量都会大幅提升。加上 AI 工具的辅助，入门门槛已经大大降低。

快速上手路线图：

  1. 安装发行版（TeX Live / MacTeX）
  2. 安装 TeXStudio
  3. 复制一个模板，按 F5 编译
  4. 遇到问题用 AI prompt 解决
  5. 逐渐掌握进阶用法

OpenClaw：开源跨平台个人 AI 助手

April 20, 2025

什么是 OpenClaw？

OpenClaw 是一个开源的个人 AI 助手项目，支持 WhatsApp、Telegram、Slack、Discord、iMessage、微信等 20+ 平台。它最大的特点是「自己的数据自己做主」—— 完全在你自己的设备上运行，不需要依赖第三方云服务。

项目亮点

- 361,000+ GitHub Stars，MIT 开源协议
- 支持 20+ 消息平台（WhatsApp、Telegram、Slack、Discord、Signal、iMessage、微信、QQ等）
- 跨平台运行（macOS / iOS / Android）
- 支持语音对话和 Canvas 可视化
- TypeScript 编写，可扩展性强

安装方式

Quickstart: 

# 克隆项目
git clone https://github.com/openclaw/openclaw.git
cd openclaw

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入你的 API 密钥

# 启动
npm run start
支持的消息平台

WhatsApp, Telegram, Slack, Discord, Google Chat, Signal, iMessage, BlueBubbles, IRC, Microsoft Teams, Matrix, 飞书, LINE, Mattermost, Nextcloud Talk, Nostr, Synology Chat, Tlon, Twitch, Zalo, WeChat, QQ, WebChat

适用场景

- 想要一个私有的 AI 助手，不想把数据交给大公司
- 需要跨多个聊天平台统一管理 AI 对话
- 开发者想要二次开发自己的 AI 助手

完
